# aquify-url-shorterner
A simple implementation for URL shortener using MERN stack technologies


Technology Stack used:
->M:MongoDB 
->E:Express
->R:ReactJS
->N:NodeJS

This website accepts the user to Register and Login to the main page where it takes in a LONG URL and generates a simple and shorter URL.
The records/history if stored in the DataBase and retirieved whenever user visits the Website.


It also counts the number of clicks the user has made on any stored link and updates the same in the database.

The Screenshots of the Website is provided under /public/images folder.
